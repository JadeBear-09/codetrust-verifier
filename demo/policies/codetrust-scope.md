# CodeTrust repository scope

## Outcome

Maintain an evidence-first verification agent for software pull requests.

## In scope

- Verification rules, GitHub pull-request ingestion, reports, dashboard, and tests.

## Out of scope

- Weather forecast dashboard features.
- Recipe recommendation features.
- Games and entertainment features.

## Acceptance criteria

- Every finding includes file, line, evidence, impact, and suggested verification.
- Deterministic evidence stays separate from model interpretation.
